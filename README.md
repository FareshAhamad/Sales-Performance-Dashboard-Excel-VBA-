# 📊 Sales Performance Dashboard (Excel + VBA)

## 📌 Overview  

This project is an **interactive Excel dashboard** built to analyze and track **sales performance across multiple cities and sales executives**.

It uses **Excel, Pivot Tables, Charts, and VBA (Macros)** to automate data processing and present insights in a clear, visual format for better decision-making.

## 🎯 Key Features  

- 📍 City-wise filtering (Chennai, Delhi, Mumbai, etc.)  
- 👨‍💼 Sales Executive performance tracking  
- 📊 Total Sales comparison (Bar Chart)  
- 🎯 Target Hit % analysis (Pie Chart)  
- 📉 Away From Target % tracking (Line Chart)  
- ⚡ Interactive dashboard navigation  
- 🔄 Automated updates using Macros
